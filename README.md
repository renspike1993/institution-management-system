# institution-management-system
Integrated systems of an institution

Systems:
1. Enrollment System (Public Institution Portal)
2. Entrance Examanition System (Guidance Department)
3. Evaluations, Grades, requirements, credentials (Registrar's Office)
4. Library Management System with OPAC with advance search engine (Library Department)
5. Attendance applicable for turnstyle gate with QR Codes and Face recognitions (Security Guards Department)
6. Billings of subject enrolled, Entrance Exams, Requests of Credentials (Finance Department)
